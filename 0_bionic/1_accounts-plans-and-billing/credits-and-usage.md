Bionic uses credits when a session runs a **Cloud** model. The amount depends on the selected model and the tokens processed.

**Credits are only consumed when you select and use cloud models.** Local models and remote LM Link models do not consume credits and do not require billing set up. 

## View usage

Open **Settings** → **Billing and Usage**.

Choose **Personal** or an available organization under **Billing account**. The data reflects how many credits your user has consumed from the selected account over the last 30 days. Organization billing and credits are only available on the web.

Select **Refresh** to load the latest account and usage information.

Bionic shows usage totals, not a fixed cost per task. Model pricing and request size affect the final spend.

To change a plan or add credits, see [Set Up Billing and Add Credits](/docs/bionic/accounts-plans-and-billing).
