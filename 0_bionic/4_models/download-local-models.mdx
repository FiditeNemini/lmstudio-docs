Downloaded models can run on your computer without using cloud inference credits.

## Find a model

1. Open **Settings**.
2. Under **Local Models**, select **Explore**.
3. Search for a model or choose one of the listed models.

Use the format filters and device-fit information to narrow the results.

## Download the model

1. Choose the download destination if more than one device is available (remote devices in LM Link).
2. Select the format and variant.
3. Start the download.

You can pause, resume, cancel, or retry a download.

## Use the downloaded model

Open **Settings** → **Local Models** → **Library** to see indexed models on the device. Then open a Bionic session and select the model from the list to use it in a session.

See [Choose a Cloud, Local, or Remote Model](/docs/bionic/models).
